## ✨ Features & Design Philosophy
This project is designed with a premium, state-of-the-art developer/tech aesthetic, following modern design principles:
- **Bento Box Layout:** Utilizes CSS Grid to structure information in a highly-organized, visually striking mosaic pattern.
- **Harmonious Color Palette:** Sleek dark-mode background (`#0a0a0f`) paired with vibrant indigo and purple gradients, glassmorphism card designs, and subtle ambient glows.
- **Micro-Animations & Hover Effects:** Smooth translations, glowing shadows on hover, and custom keyframe animations (like pulse badges) to create a page that feels responsive and alive.
- **Fully Responsive:** Optimized with custom CSS media queries for desktop, tablet, and mobile viewing.
- **Enterprise-ready UI:** Includes section highlights for enterprise rewards, partner integration logos, statistics tracker, API code snippet blocks, and lead-generation coverage forms.

## 🛠️ Technology Stack
- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (CSS Variables, Grid, Flexbox, Keyframe animations)
- **Typography:** Google Fonts (*Inter* & *Space Grotesk*)
- **Icons:** SVG-based icons

## 🚀 Getting Started Locally

To run this project locally, simply clone the repository and open `index.html` in any web browser, or use a local static file server.

### Option 1: Live Server (recommended for development)
If using VS Code, install the **Live Server** extension, right-click `index.html` and select **Open with Live Server**.

### Option 2: Simple HTTP Server
Using Python:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`.
